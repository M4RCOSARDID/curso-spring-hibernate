## Bloque A: Infraestructura Docker

### Momento Arranque
Al intentar arrancar la aplicacion y poder visualizar datos no pude.

### Momento Error
Cuando fui a arrancar el docker, este no me dejaba y era porque el puerto ya estaba siendo usado por otro y tuve que cambiarlo.

### Momento Aprendizaje
Aprendí a como ver un error y saber analizar antes de ejecutar.

## Bloque B: API REST Spring Boot

### Momento Arranque
Cuando intente compilar la aplicacion esta no lo hacía. 

### Momento Error
Resultó que me daba un error en el POM con las dependencias, las tenia mal declaradas las de SPRING BOOT, al ponerlas bien si que funcionaba

### Momento Aprendizaje
Aprendi a hacer bien el POM.

## Bloque C: Funcionalidad avanzada

### Momento Arranque
Al intentar validar datos no podia.

### Momento Error
Intenté validar datos y no lo hacia correctamente, se solucionó reescribiendo el código y pidiendo ayuda a la IA para que detectase que estaba mal.

### Momento Aprendizaje
Aprendí a usar la Ia para detectar errores pero tambien entender el porque fueron y como solucionarlos
